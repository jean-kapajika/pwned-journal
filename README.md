# pwned-journal
A centralized hub for documenting and sharing my cybersecurity practice journey

# Purpose 
This repository serves as a central location to organize my notes and resources for the machines I work on.
The goal is to document my learning process, track progress, and share knowledge with others in the future. It's intended for personal use initally, but could be expanded to a more public resource later. 

# Project Structure
To facilitate navigation and retrieval of information, I'm using the following structure:
- **Platform_Name**: 
	- **Machine_Name**: This folder contains notes and subfolders dedicated to a single machine. 
### Naming Convention
I will use the official names of the machines I work on for the directory within **Platform_name**.
